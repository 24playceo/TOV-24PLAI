# Веб-сайт TOV 24PLAI

## Структура проекту

```
website-TOV-24PLAI/
├── index.html              # Головна сторінка
├── privacy.html            # Сторінка політики конфіденційності
└── css/
    └── style.css           # Стилі для всього сайту
```

## Доступ до сторінок

- **Головна сторінка**: відкрийте `index.html`
- **Політика конфіденційності**: відкрийте `privacy.html` або перейдіть через меню навігації

## Як відкрити сайт

1. Відкрийте файл `index.html` у браузері
2. Або запустіть локальний веб-сервер в цій папці

### Використання локального сервера (рекомендовано)

**Python:**
```bash
python -m http.server 8000
```

**Node.js (якщо встановлено http-server):**
```bash
npx http-server
```

Потім відкрийте у браузері: `http://localhost:8000`

## Можливості сайту

- ✅ Адаптивний дизайн (працює на мобільних пристроях)
- ✅ Сучасний інтерфейс
- ✅ Сторінка політики конфіденційності відповідно до вимог законодавства України
- ✅ Навігаційне меню
- ✅ Підвал з посиланням на політику конфіденційності

## Редагування

- Щоб змінити контент, редагуйте HTML файли
- Щоб змінити стилі, редагуйте `css/style.css`
- Не забудьте оновити контактну інформацію в розділі "Контактна інформація" на сторінці політики конфіденційності
